Проект: Предсказание стартапов
Цель проекта: разработать модель машинного обучения для предсказания продолжения деятельности стартапа. А так же провести полноценный разведочный анализ и сформировать рекомендации будущим создателям стартапов (какие факторы влияют на успешность стартапа).

Этапы проекта:

Открыть файлы с данными.
Провести предобработку данных.
Провести исследовательский анализ данных.
Провести корреляционный анализ признаков данных.
Обучить несколько моделей и выбрать лучшую.
Оценить важность признаков.
